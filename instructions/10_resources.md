## Models (HuggingFace Hub)
- **roberta-base**: https://huggingface.co/roberta-base (recommended)
- **distilbert-base-uncased**: https://huggingface.co/distilbert-base-uncased (faster, less accurate)
- **bert-base-uncased**: https://huggingface.co/bert-base-uncased (older, still good)
- **Pre-trained Fake News Model** (for reference): https://huggingface.co/hamzab/roberta-fake-news-classification

## Key Libraries
| Library | Use | Install |
|---|---|---|
| `transformers` | HuggingFace models | `pip install transformers` |
| `datasets` | HuggingFace dataset loading | `pip install datasets` |
| `torch` | Deep learning backend | `pip install torch` |
| `accelerate` | HuggingFace training acceleration | `pip install accelerate` |
| `scikit-learn` | Baseline models, metrics | `pip install scikit-learn` |
| `pandas` | Data manipulation | `pip install pandas` |
| `gradio` | Demo UI | `pip install gradio` |
| `fastapi` | API deployment | `pip install fastapi uvicorn` |
| `matplotlib` | Plots | `pip install matplotlib` |
| `seaborn` | Better plots | `pip install seaborn` |
